sidebarTitle: Checkpointers
description: "Integrate with checkpointer backends for LangGraph persistence."
---

Checkpointers enable [persistence](/oss/python/langgraph/persistence) in LangGraph, allowing agents to save and resume state across interactions.

| Backend | Package |
|---------|---------|
| In-memory | [`langgraph-checkpoint`](https://pypi.org/project/langgraph-checkpoint/) |
| SQLite | [`langgraph-checkpoint-sqlite`](https://pypi.org/project/langgraph-checkpoint-sqlite/) |
| PostgreSQL | [`langgraph-checkpoint-postgres`](https://pypi.org/project/langgraph-checkpoint-postgres/) |
| MongoDB | [`langgraph-checkpoint-mongodb`](https://pypi.org/project/langgraph-checkpoint-mongodb/) |
| Redis | [`langgraph-checkpoint-redis`](https://pypi.org/project/langgraph-checkpoint-redis/) |
| Azure Cosmos DB | [`langgraph-checkpoint-cosmosdb`](https://pypi.org/project/langgraph-checkpoint-cosmosdb/) |
| AWS (DynamoDB, Bedrock, Valkey) | [`langgraph-checkpoint-aws`](https://pypi.org/project/langgraph-checkpoint-aws/) |
